# gacha_sim

## 权限与注册边界

- 管理员命令权限为 `ADMIN_PERMISSION`（超级用户或 `user.is_admin=True`）。
- 当前实际注册的 handler 见 `src/plugins/handler/__init__.py`：`admin`、`checkin`、`gacha_template`、`gacha_item`、`gacha_banner`、`gacha_history`、`group_settings`。

## 管理员与群设置指令

- `设置管理员#<qq>`（管理员）
  - 边界：`qq` 必须为纯数字。
- `删除管理员#<qq>`（管理员）
  - 边界：目标必须已是管理员。
- `列出管理员`（管理员）
  - 边界：无管理员时返回“暂无管理员”。
- `开启群[#<群号>]`（管理员）
  - 边界：私聊必须带群号；群聊默认当前群。
- `关闭群[#<群号>]`（管理员）
  - 边界：私聊必须带群号；群聊默认当前群。
- `设置默认模板#<模板名>[#<群号>]`（管理员）
  - 边界：模板必须存在；私聊需显式群号；群聊默认当前群。
- `设置默认时区#<UTC偏移>`（管理员）
  - 示例：`设置默认时区#0`、`设置默认时区#+8`、`设置默认时区#-5`。
  - 边界：仅接受整数偏移（范围 `-12~+14`）；会写入 `config.default_timezone`。
- `查看群设置#<群号>`（管理员）
  - 边界：私聊必须显式群号；群聊默认当前群。

## 模板与卡片指令

- `创建卡池模板#<模板名>#<描述>`（管理员）
  - 交互输入稀有度配置：每行 `稀有度#概率`。
  - 边界：概率必须为整数且总和为 100。
- `删除卡池模板#<模板名>`（管理员）
  - 边界：模板不存在会失败。
- `列出卡池模板`（管理员）
- `添加卡片#<模板名>#<卡片名>#<稀有度>#<描述>`（管理员）
  - 边界：模板与稀有度必须存在；同模板下卡片名唯一。
- `删除卡片#<模板名>#<卡片名>`（管理员）
  - 边界：模板/卡片不存在会失败。
- `列出卡片#<模板名>`（管理员）
  - 边界：模板不存在或无卡片会提示。

## 卡池运行与抽卡指令

- `创建卡池#<模板名>#<Banner名>[#<描述>]`（描述可为空。管理员）
  - 交互输入：开启时间、结束时间（`无`=永久开放）、UP 配置（每行 `稀有度#up_share#id1,id2`，`无`=不配置）。
  - 时间语义：开启/结束时间按“默认时区”解析，再转换为 UTC 存储。
  - 边界：同模板下 `Banner名` 唯一；若已存在同名卡池会直接提示使用 `开启复刻#<模板名>#<Banner名>`。
- `开启复刻#<模板名>#<Banner名>`（管理员）
  - 交互输入：开启时间、结束时间。
  - 边界：若该卡池当前存在运行中的记录，则不允许开启复刻。
  - 边界：同一 `banner_pool_id` 运行区间禁止重叠（含与永久开放区间冲突）。
- `查看开放卡池`（管理员）
  - 边界：仅群聊；依赖当前群默认模板；仅展示当前开放中的运行记录。
- `抽卡#<Banner名>[#十连]`（所有用户）
  - 边界：仅群聊；以当前群默认模板为作用域；仅命中当前开放 run。
  - 边界：第三段参数严格仅支持 `十连`，留空表示单抽；例如 `抽卡#桃花妖up池#十连`。
  - 消耗：按“每抽所需积分”扣除积分（十连=10倍）；积分不足会拒绝抽卡。
  - 边界：若命中多个开放 run（异常数据）会拒绝抽取并提示修复。
  - 返回：单抽返回 1 条结果；十连返回 10 条明细（同一批次记录同一 `draw_date`，`draw_index=1..10`，并记录每条掉落的积分消耗）。

## 抽卡记录与统计指令

- `抽卡记录#[N]`（所有用户）
  - 边界：`N` 可选，未填写默认 `30`；取值范围 `1~50`，越界会提示输入合法数值。
  - 展示时间：按“默认时区”显示。
  - 边界：无记录时返回“暂无抽卡记录”。
- `抽卡统计`（所有用户）
  - 边界：无记录时返回“暂无抽卡统计”。
  - 返回：按卡池聚合的总掉落与卡片 Top 明细。

## 积分与余额指令

- `签到`（所有用户）
  - 边界：同一用户每天一次（按“默认时区”自然日判定）；奖励区间来自配置，默认 `5-10`；奖励发放到积分。
- `设置签到积分#<min-max>`（管理员）
  - 边界：格式必须为正整数区间且 `min<=max`。
- `设置利率#<rate>`（管理员）
  - 边界：范围 `0~100`（单位：%/天）。
- `设置抽卡积分#<cost>`（管理员）
  - 边界：`cost` 必须为正整数，表示每抽消耗积分。
- `领取利息`（所有用户）
  - 边界：存款>0、利率>0、距上次领取满 1 天；利息发放到积分。
- `查询余额`（所有用户）
  - 展示：固定两行，显示当前存款、当前积分与当前利率。
- `存款#<数量>`（所有用户）
  - 边界：数量必须为正整数；只影响存款本金。
- `取款#<数量>`（所有用户）
  - 边界：数量必须为正整数且存款充足；只影响存款本金。
- `转让积分#<目标QQ>#<数量>`（所有用户）
  - 边界：目标 QQ 格式合法、不能转给自己、数量为正；需二次确认（确认/取消）。
- `管理加积分#<目标QQ>#<数量>`（管理员）
  - 边界：目标 QQ 合法、数量为正。
- `管理扣积分#<目标QQ>#<数量>`（管理员）
  - 边界：目标 QQ 合法、数量为正、目标积分充足。
