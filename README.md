# Gacha Sim 使用说明

## 功能简介

本项目是基于 NoneBot2 的群聊抽卡模拟插件，提供：

- 卡池模板 / 稀有度 / 卡片管理
- 卡池创建、复刻、开放查询与抽卡
- 抽卡历史与统计
- 积分签到、存取款、利息、积分转让
- 场景（群）开关、默认模板、默认时区等配置

> 💡 `菜单` 命令会将本 README 渲染为图片并发送。
> 💡 私聊是否可用取决于 `设置私聊互动#开/关`。

---

## 快速上手

```text
1. 设置管理员#123456789                    ← 可选，添加其他管理员
2. 开启群                                  ← 在目标群中开启功能
3. 创建卡池模板#标准池#默认模板
4. （按提示输入）SSR#2 / SR#18 / R#80       ← 稀有度概率总和需 100
5. 添加卡片#标准池#刻晴#SSR#雷系角色
6. 设置默认模板#标准池
7. 创建卡池#标准池#新手池#常驻测试
8.（按提示输入）开启时间 / 结束时间 / UP配置
9. 签到                                   ← 获取积分
10. 抽卡#新手池#十连
```

## 交互输入格式

### 创建卡池模板

命令：`创建卡池模板#<模板名>#<描述>` 后，按行输入：

```text
稀有度#概率
```

示例：

```text
SSR#2
SR#18
R#80
```

约束：概率必须是整数，且总和必须等于 `100`。

### 创建卡池 / 开启复刻

1) 开启时间：`YYYY-MM-DD HH:MM`
2) 结束时间：`YYYY-MM-DD HH:MM` 或 `无`（常驻）
3) 仅创建卡池需要输入 UP 配置，格式为：

```text
稀有度#up_share#卡片名1,卡片名2
```

示例：

```text
SSR#50#刻晴,莫娜
SR#70#香菱
```

输入 `无` 表示不配置 UP。

## 消息生效条件

以下条件全部满足时，相关命令才会正常执行：

1. ✅ 发送者满足命令权限（管理员命令需超级管理员或管理员）
2. ✅ 命令格式正确（参数数量、分隔符 `#`、参数类型）
3. ✅ 若命令绑定了 `IS_GROUP_ENABLE`：
    - 群聊场景下该群已开启
    - 私聊场景下已开启 `设置私聊互动#开`
4. ✅ 业务数据满足约束（如模板存在、积分足够、时间区间合法）

## 默认值与规则

- 默认时区：`UTC+8`
- 默认签到积分区间：`5-10`
- 默认利率：`0`（%/天）
- 默认每抽积分消耗：`1`
- 抽卡记录默认条数：`30`
- 抽卡记录条数范围：`1~50`

## 常见问题

| 问题              | 可能原因                      |
|-----------------|---------------------------|
| 抽卡提示“未开放”或无结果   | 当前群未设置默认模板，或目标卡池当前不在开放时间内 |
| 群里命令没反应         | 该群可能未开启（先执行 `开启群`）        |
| 私聊命令不可用         | 未开启 `设置私聊互动#开`            |
| `抽卡#xxx#abc` 失败 | 第三个参数仅支持 `十连`，不填为单抽       |
| `签到` 失败提示已签到    | 按默认时区自然日判定，一天仅一次          |
| 领取利息失败          | 存款为 0、利率为 0，或距离上次领取未满 1 天 |
