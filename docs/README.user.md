# 用户菜单

欢迎使用抽卡模拟机器人！你可以在这里完成抽卡、查看记录统计，以及通过签到和钱包功能管理积分。
建议新用户先按这个顺序体验：`签到` -> `查看开放卡池` -> `抽卡#<卡池名>#十连`。

> 💡 私聊是否可用取决于管理员是否开启

---

### 抽卡与记录

| 命令                                      | 说明                            |
|-----------------------------------------|-------------------------------|
| `查看开放卡池`                                | 查看当前正在开放的卡池                   |
| `抽卡#<卡池名>[#十连]` 或 `抽卡#<模板名>#<卡池名>[#十连]` | 群聊可指定模板，私聊必须指定模板<br/>可以填写十连参数 |
| `抽卡记录#[N]`                              | 查看最近 N 条记录（默认 30，范围 1~50）     |
| `抽卡统计`                                  | 按卡池聚合统计掉落                     |

### 积分与钱包

| 命令                 | 说明             |
|--------------------|----------------|
| `签到`               | 每日签到领取随机积分     |
| `领取利息`             | 领取存款利息（发放为积分）  |
| `查询余额`             | 查看存款、积分、利率     |
| `存款#<数量>`          | 增加存款           |
| `取款#<数量>`          | 提取存款           |
| `转让积分#<目标QQ>#<数量>` | 向其他用户转让积分（需确认） |

### 帮助

| 命令   | 说明           |
|------|--------------|
| `菜单` | 将说明文档渲染为图片发送 |

---

## 参数与规则

- 群聊支持：`抽卡#<卡池名>[#十连]`（默认模板）或 `抽卡#<模板名>#<卡池名>[#十连]`（显式模板）。
- 私聊仅支持：`抽卡#<模板名>#<卡池名>[#十连]`；不支持仅传卡池名。
- 兼容旧语法：`抽卡#<卡池名>#十连` 仍可用（群聊默认模板）。
- `抽卡记录#[N]`：`N` 默认 `30`，范围 `1~50`。
- 签到按默认时区自然日判定，一天仅一次。

## 常见问题

| 问题                  | 可能原因                             |
|---------------------|----------------------------------|
| 抽卡提示“未开放”或无结果       | 当前群未设置默认模板，或目标卡池当前不在开放时间内        |
| 群里命令没反应             | 该群可能未开启（联系管理员执行 `开启群`）           |
| 私聊命令不可用             | 未开启 `设置私聊互动#开`                   |
| 私聊中 `抽卡#<卡池名>` 失败   | 私聊必须显式指定模板：`抽卡#<模板名>#<卡池名>[#十连]` |
| `抽卡#xxx#yyy#abc` 失败 | 第三个参数仅支持 `十连`                    |
| `签到` 失败提示已签到        | 按默认时区自然日判定，一天仅一次                 |
| 领取利息失败              | 存款为 0、利率为 0，或距离上次领取未满 1 天        |
