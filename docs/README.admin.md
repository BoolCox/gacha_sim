# Gacha Sim 管理员说明

## 功能总览

- 管理员增删与列表
- 群开关、默认模板、默认时区、私聊互动配置
- 卡池模板 / 稀有度 / 卡片管理
- 卡池创建与复刻（开放时间、UP 配置）
- 抽卡统计、钱包参数、管理加减积分

---

## 权限说明

| 角色   | 来源                               | 可执行命令        |
|------|----------------------------------|--------------|
| 管理员  | 由管理员通过 `设置管理员#<QQ>` 赋权，初始管理员不可删除 | 所有命令         |
| 普通用户 | /                                | 抽卡、签到、余额、统计等 |

## 管理命令一览

### 管理员管理

| 命令           | 说明      |
|--------------|---------|
| `设置管理员#<QQ>` | 添加管理员   |
| `删除管理员#<QQ>` | 删除管理员   |
| `列出管理员`      | 查看管理员列表 |

### 场景与系统设置

| 命令                    | 说明                        |
|-----------------------|---------------------------|
| `开启群[#<群号>]`          | 开启某群功能（群聊默认当前群）           |
| `关闭群[#<群号>]`          | 关闭某群功能（群聊默认当前群）           |
| `设置默认模板#<模板名>[#<群号>]` | 设置群默认模板                   |
| `设置默认时区#<UTC偏移>`      | 设置全局默认时区（`-12` 到 `+14`）   |
| `设置私聊互动#开/关`          | 控制私聊是否允许通过规则校验            |
| `查看配置[#<群号>]`         | 查看群状态、默认模板、当前利率、默认时区、私聊互动 |

### 模板、卡池与卡片管理

| 命令                            | 说明                |
|-------------------------------|-------------------|
| `创建卡池模板#<模板名>#<描述>`           | 创建模板；随后交互输入稀有度概率  |
| `删除卡池模板#<模板名>`                | 删除模板              |
| `创建卡池#<模板名>#<卡池名>[#描述]`       | 创建卡池并交互录入时间、UP 配置 |
| `开启复刻#<模板名>#<卡池名>`            | 对已有卡池开启新一轮时间区间    |
| `列出卡池模板`                      | 查看模板列表            |
| `添加卡片#<模板名>#<卡片名>#<稀有度>#<描述>` | 向模板添加卡片           |
| `删除卡片#<模板名>#<卡片名>`            | 从模板删除卡片           |
| `列出卡片#<模板名>`                  | 查看模板下卡片           |

### 经济系统管理

| 命令                  | 说明         |
|---------------------|------------|
| `设置签到积分#<min-max>`  | 设置签到积分区间   |
| `设置利率#<rate>`       | 设置日利率（%/天） |
| `设置抽卡积分#<cost>`     | 设置每抽积分消耗   |

## 创建流程

### 创建卡池模板

命令：`创建卡池模板#<模板名>#<描述>` 后，按提示输入：

```text
稀有度#概率
```

示例：

```text
SSR#2
SR#18
R#80
```

约束：概率必须是整数，且总和必须等于 `100`。

### 创建卡池 / 开启复刻

1) 开启时间：`YYYY-MM-DD HH:MM`
2) 结束时间：`YYYY-MM-DD HH:MM` 或 `无`（常驻）
3) 仅创建卡池需要输入 UP 配置，格式：

```text
稀有度#<UP卡片概率>#卡片名1,卡片名2
```

> 💡 UP卡片概率 = 稀有度基础概率 * (`<up_share>` / 100)。

示例：

```text
SSR#50#刻晴,莫娜
SR#70#香菱
```

输入 `无` 表示不配置 UP。

## 默认值与规则

- 默认时区：`UTC+8`
- 默认签到积分区间：`5-10`
- 默认利率：`0`（%/天）
- 默认每抽积分消耗：`1`
- 抽卡记录默认条数：`30`
- 抽卡记录条数范围：`1~50`

## 管理员常见排查

| 问题              | 可能原因                      |
|-----------------|---------------------------|
| 用户抽卡提示“未开放”或无结果 | 群未设置默认模板，或卡池当前不在开放时间内     |
| 群里命令没反应         | 群未开启（执行 `开启群`）            |
| 私聊命令不可用         | 未开启 `设置私聊互动#开`            |
| 用户签到失败提示已签到     | 按默认时区自然日判定，一天仅一次          |
| 用户领取利息失败        | 存款为 0、利率为 0，或距离上次领取未满 1 天 |
